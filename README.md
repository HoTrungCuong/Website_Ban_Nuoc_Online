# Website Bán Nước Online

## Giới thiệu
Website bán nước online được xây dựng bằng ASP.NET MVC 5 và SQL Server, hỗ trợ bán các loại đồ uống và quản lý cửa hàng.

## Công nghệ sử dụng
- ASP.NET MVC 5
- C#
- Entity Framework
- SQL Server LocalDB
- HTML, CSS, Bootstrap
- JavaScript

## Chức năng

### Khách hàng
- Đăng ký
- Đăng nhập
- Xem danh sách sản phẩm
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Thêm vào giỏ hàng
- Đặt hàng
- Nạp tiền vào tài khoản
- Xem lịch sử đơn hàng

### Quản trị viên
- Quản lý sản phẩm
- Quản lý loại sản phẩm
- Quản lý khách hàng
- Quản lý đơn hàng
- Quản lý khuyến mãi
- Quản lý nhân viên

```text
Controllers/

```

## Hướng dẫn chạy

1. Clone repository:

```bash
git clone https://github.com/HoTrungCuong/Website_Ban_Nuoc_Online.git
```

2. Mở bằng Visual Studio 2022.

3. Restore NuGet Packages.

4. Cấu hình SQL Server LocalDB.

5. Chạy bằng IIS Express.
